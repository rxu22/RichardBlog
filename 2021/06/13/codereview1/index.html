Code Review 6/10

This method prints out asteriks in the shape of a square with a length and width of n

Since this method is supposed to print out an asterik square, the row and col should iterate the same amount of times.
“\n” inserts a new line

This method prints out a right triangle.

Similar to the last method, the number of rows follow the same procedure. This time, however, the number of columns is equal to n - the number of rows.

This method prints out a nicer triangle

Changed the rows to match the number n. The columns are odd numbers so the code uses 2 * the current row number - 1. Since the shape of this triangle is different, I added a for loop to add the matching number of spaces.

This method finds the max number of a given array

First created an integer value called “max” and set it to 0. Created a for loop to read the entire array, and if the given index of the array is greater than the max value, the max value changes to the given index.

The other for loops works for negative values since the max value is set equal to Integer.MIN_VALUE which is the smallest number even with negatives considered.
