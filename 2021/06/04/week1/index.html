Week 1 CS

Learned about html and markdown and began exploring the CS 61A class.
For html, I followed the website: Tutorial Point
As for markdown, I looked at this cheat sheet: Markdown Cheatsheet

I followed the tutorial tabs under the “HTML Tutorial”. Html was very easy to follow as the tutorialpoint website gave very concrete examples on how formatting should look like, and how the code will turn out.
Markdown was a much easier learning process for me since I already learned the basics of html at this point.

After learning these two front end languages, I proceeded to follow the CS 61A from Berkley CS 61A
I followed its week 1 schedule by looking at their slides, reading the online textbook (chapters 1.1-1.4), and working on their lab assignment for the week (coded in python)

The first lesson was very simple as the slides and textbook gave a broad overview of computer science. It went over what was going to be covered in the course and an intro to python values (integers, booleans, strings, etc.). These were pretty simple to understand as I already had prior knowledge to this.

The second part of week 1 (chapters 1.2-1.4) started showing actual coding examples in python. They showed mathmatical operators, how to write functions, and proper syntax. I found this part much more fun to learn overall.

Lastly for week 1, I worked on the lab assignment. It required me to install python, and visual code. I followed the steps outlined in the Lab instructions and used several commands (ls, cd, cd.., mkdir) in windows powershell to create a new folder for this lab and extract the zipped files. Since I did not have a account I just took notes on what the answers to the ok test were:

  1. 10 + 2 = 12
  2. 7 / 2 = 3.5
  3. 7 // 2 = 3
  4. 7 % 2 = 1
