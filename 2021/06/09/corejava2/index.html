3.5 Operators

Operators are used to combine values

3.5.1 Arithmetic Operators

  • +,-,*,/
  • / denotes integer division if both arguments are integers while % denotes modulus division
  • integer division by 0 raises an exception
  • floating-point division by 0 yields an infinite or NaN result

3.5.3 Conversions between Numeric Types

  • certain numeric types can be converted to different types, however, there may be some precision lost
  • Some have certain rules:
  1. If either of the operands is of type double, the other one will be converted to a double
  2. If either of the operands is of type float, the other one will be converted to a float.
  3. If either of the operands is of type long, the other one will be converted to a long.
  4. Otherwise, both operands will be converted to an int

3.8 Control Flow

3.8.1 Block Scope

  • Cannot declare identically named variables in two nested blocks

3.8.2 Conditional Statements

  • if, else (optional)

3.8.3 Loops

  • while loop executes a statement if the condition is true
  • chance that while loop never executes. Use do while loop to execute at least once

3.8.4 Determinate Loops

  • for loop supports iteration with a counter
  • 1st slot: usually the counter initialization ex: int i = 1
  • 2nd skot: gives the condition that will be tested before each new pass through the loop ex: i <= 10
  • 3rd slot: how to update the counter ex: i++
  • When you declare a variable in the first slot of the for statement, the scope of that variable extends until the end of the body of the for loop
  • if you define a variable inside a for statement, you cannot use its value outside the loop
  • you can define variables with the same name in separate for loops

3.8.5 The Switch Statement

  • case label can be: a constant expression of type char, byte, short, or int, an enumerated constant, or a string literal

3.8.6 Statements That Break Control Flow

  • break statements can also break out of loop
  • continue statements transfers control to the header of the innermost enclosing loop
  • goes back to the loop header
  • if continue is used in a for loop, it goes back to the update part of the loop
  • If n < 0, then the continue statement jumps to the count++ statement.
