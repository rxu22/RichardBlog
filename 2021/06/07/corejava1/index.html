3.1 Simple Java Program

Java is case sensitive, so the system will not run if there are mistakes in capitalization
Given the code:

  • Main has to be “main”, or the program will not compile

  • The access modifier sets the accessibility of classes

  • Naming a class: Must begin with a letter, and can be followed by any combination of letters/numbers

3.3 Data Types

3.3.1 Integer Types

  • int
  • short
  • long
  • byte

3.3.2 Floating-Point Types

  • float (6-7 significant decimal digits)
  • double (15 significant decimal digits)
  • used for numbers with fractional parts
  • Double means that it has twice the precision of float type
  • Float types have the suffix F or f, so floating types without F or f are considered as the type double

3.3.3 Char Type

  • use ‘xxx’ to display certain characters
  • The same character can be denoted with the matching number

3.3.5 Boolean Type
has two values: true or false
used for evaluating logical conditions
cannot convert between integers and boolean values

3.4 Variables and Constants

3.4.1 Declaring Variables

  • Declare variables by placing the type first followed by the name of the variable
  • ex: double salary;
  • Semicolons are necessary since a declaration is a complete java statement, and all java statements end in a semicolon
  • A variable name must begin with a letter
  • cannot use a java reserved word as a variable name
  • To declare multiple variables in the same line: ex - int i, j; // both are integers
